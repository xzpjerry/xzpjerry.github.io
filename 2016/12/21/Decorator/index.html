Decorator is useful when using others’ code.

Especially when you wanna add some function, say logging, to it.

To illustrate, we need a sample function first.

What if we wanna add logging function so that it can tell us the date when it excuate?

Basically, it means using another function, in order to add new funtion, to call target function.

It’s like”log(func) -> wrapper() -> func()”

We can also custom log from outside.

Why not be simpler?

Put ‘@funcB’ above ‘funcA’ means ‘funcA = funcB(funcA)’
